# 📦 CarryChest - Free Skript


--  

  
Tired of moving your items back and forth? **CarryChest** is a lightweight, zero-lag Skript that allows players to pick up Chests and Barrels (along with their entire inventory) and carry them on their heads!

🌟 **Looking for more?** If you want to carry **Mobs (Villagers, Animals)** and other decorative blocks (Anvils, Spawners, Glass, etc.), check out our **Premium Version**: [AdvancedCarry on BuiltByBit](link)

## ✨ Features
- **Carry Full Containers**: Pick up Chests, Barrels, and Trapped Chests without dropping a single item.
- **Zero Lag (TPS Friendly)**: Uses Minecraft's native "Passenger" mounting system combined with Block Displays. Absolutely no heavy teleportation loops!
- **100% Dupe & Glitch Protected**: If a player dies or disconnects while carrying a chest, the chest will automatically and safely be placed at their last location. Zero item loss, zero duplication bugs!
- **Respects Protections**: Players cannot pick up or place chests in protected areas (Works with WorldGuard, GriefPrevention, Lands, etc.).


## 📖 How to Use
1. Look at a Chest or Barrel with an **Empty Hand**.
2. **Sneak (Shift) + Left Click** to pick it up onto your head.
3. Run to your new house!
4. **Sneak (Shift) + Right Click** on the ground to place it down safely.

### Commands:
- `/carrychest reset` (or `/cc reset`) - Resets the player's carry state in case of any visual/movement glitches.

## 📥 Installation
1. Download `carrychest.sk`.
2. Place the file into your server's `plugins/Skript/scripts/` folder.
3. Type `/sk reload carrychest` in your server console or in-game.
4. Enjoy!

## ⚙️ Requirements
- Minecraft 1.19.4 - 1.21+ (Requires Block Display entity support)
- [Skript](https://github.com/SkriptLang/Skript) (Latest version recommended)
